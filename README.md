# 📖 三代經課 (RCL) 中文版百年日曆生成器

這是一個開源的 Python 工具，旨在將英文版的修訂通用經課表 (Revised Common Lectionary, RCL) 原始 ICS 檔案，自動擴充、翻譯並推算生成未來 100 年的「中文版三代經課 Google 日曆」。

## ✨ 核心特色
* **百年曆法推算**：基於 2025-2028 的種子檔案，自動推算未來百年（甲、乙、丙年）的復活節、將臨期等浮動節期。
* **全中文翻譯**：包含節期名稱、書卷名稱的自動翻譯，並修正外語語法。
* **經文與詩歌配對**：自動為每個主日配對對應的 RCL 經課範圍，以及為華人教會量身打造的專屬推薦詩歌。
* **Google 日曆友善**：內建自動切割功能，每 15 個「教會年」切分為一個小檔案，完美避開 Google 日曆 1MB 的匯入限制。
* **防呆與補漏機制**：內建「基督君王主日」強制校正，以及各季節 Proper 主日的自動補缺。

## 🚀 如何使用 (生成百年日曆)
1. 確保您的電腦已安裝 Python 3。
2. 安裝必要的套件：
   ```bash
   pip install icalendar
   ```
3. 下載本專案的 `convert_lectionary.py` 與 `lectionary.ics` 到同一個資料夾。
4. 在終端機執行程式：
   ```bash
   python convert_lectionary.py
   ```
5. 程式會自動產出 7 個分批的 `.ics` 檔案以及一份 `generation_report.txt` 體檢報告。
6. 前往 Google 日曆的「設定 > 匯入與匯出」，將這 7 個檔案分批匯入至您的專屬日曆中。

## 🧹 附贈工具：一鍵清空日曆腳本 (Google Apps Script)
如果您在匯入過程中發生錯誤，或者想要完全重置您的「三代經課」日曆，本專案提供了一個強大的大掃除腳本 `clear_calendar.gs`，能安全避開 Google 執行時間限制，快速清空百年資料。

**使用方式：**
1. 前往 [Google Apps Script](https://script.google.com/) 並點擊「新專案」。
2. 將 `clear_calendar.gs` 裡的所有程式碼複製，貼到網頁編輯器中。
3. 找到程式碼最下方的 `targetCalendarId` 變數，將其替換為您想要清空的 Google 日曆 ID（可在 Google 日曆設定中找到）。
4. 在上方工具列選擇 `runClearTask` 函式，點擊「執行」。
5. *(註：若日曆資料龐大，腳本會自動在接近 6 分鐘限制時安全暫停，此時只需再次點擊「執行」即可接續清理。)*

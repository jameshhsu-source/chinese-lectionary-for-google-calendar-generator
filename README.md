# 📖 供Google日曆使用的中文三代經課檔生成器 (Chinese Lectionary for Google Calendar Generator)

這是一個基於 Python 的供Google日曆使用的中文三代經課檔生成器。透過擷取（2025-2027）三個教會年的英文版三代經課，自動為您生成未來 100 年的完整中文三代經課ics檔，供Google日曆使用。

## 🧰 專案包含的重要資源

* **`convert_lectionary.py`**: 核心生成主程式。
* **`lectionary.ics`**: 來源原始檔，2025-2027三個教會年(A、B、C年)的英語三代經課。
* **`clear_calendar.gs`**: Google日曆專屬清空工具 (Google Apps Script)。

---

## 🚀 快速開始 (Quick Start)

### 1. 安裝與執行主程式
1. 請確保您的電腦已安裝 Python 3。
2. 安裝必要的套件：
   `pip install icalendar`
3. 執行主程式：
   `python convert_lectionary.py`
4. 程式執行完畢後，會自動產出多份 `lectionary_cht_年份_年份.ics` 檔案（每 15 個教會年切成一包），並產出一份 `generation_report.txt` 檢驗報告。

### 2. 匯入 Google 日曆 (⚠️ 極度重要)
為了避免未來管理混亂，**強烈建議不要將 .ics 檔案直接匯入您的「個人主日曆」**！
1. 進入 Google 日曆網頁版設定。
2. 點擊左側「新增日曆」>「建立新日曆」（例如命名為：`三代經課`）。
3. 點擊左側「匯入與匯出」>「匯入」。
4. 選擇產出的 `.ics` 檔案，**並且在「新增至日曆」的下拉選單中，務必選擇您剛剛建立的 `三代經課`**。
5. 將生成的 ICS 檔案依序匯入即可。

## 🧹 Google 日曆清理指南 (Cleanup Guide)

如果您匯入失敗、想要更新版本，或是想要重置整個「三代經課」日曆，請使用本專案附帶的 `clear_calendar.gs` 程式。

🛑 **【極度危險警告】** 🛑
**此程式會無差別「清空」指定日曆中 2025 年到 2130 年的所有事件！絕對不可以填入您的個人主日曆 ID，否則您的私人行程將被全部刪除且無法復原！** 本工具僅限用於清空「專門為經課表建立的獨立日曆」。

### 如何使用清理工具？
這是一個 Google Apps Script 腳本，**不需要安裝任何軟體**，直接在瀏覽器中即可執行：

1. **取得目標日曆 ID**：
   * 打開 Google 日曆網頁版，點擊右上角「齒輪 (設定)」。
   * 在左側選單點擊您要清空的日曆（例如 `三代經課`）。
   * 往下滑找到「整合日曆」區塊，複製 **「日曆 ID」**（通常是一長串亂碼加上 `@group.calendar.google.com`）。
2. **建立腳本**：
   * 前往 [Google Apps Script](https://script.google.com/) 並登入帳號，點擊左上角「新專案」。
   * 將編輯器中的預設程式碼刪除，並將本專案 `clear_calendar.gs` 檔案裡面的所有程式碼**複製並貼上**。
3. **替換日曆 ID**：
   * 找到程式碼最下方的 `runClearTask()` 函式。
   * 將 `targetCalendarId` 後面的引號內容，**替換成您剛剛複製的日曆 ID**。
4. **執行與授權**：
   * 點擊上方的 **「儲存 (磁碟片圖示)」**。
   * 確認上方下拉選單選擇的是 `runClearTask`，然後按下 **「執行 (Run)」**。
   * *首次執行需授權*：Google 會跳出「需要授權」視窗。請點擊「審查權限」> 選擇帳號 > 點擊左下角「進階」>「前往至未命名的專案 (不安全)」>「允許」。
5. **安全暫停機制 (⚠️ 注意)**：
   * 為了避免觸發 Google 的 6 分鐘執行時限，本程式內建 **5.5 分鐘安全暫停機制**。
   * 如果您的日曆事件非常多，下方的「執行記錄」可能會顯示：`⚠️ 快要觸發 Google 6分鐘限制了！已安全暫停。請再次點擊「執行」來接續刪除！`

   * 遇到此提示時，**請再次按下上方的「執行」按鈕**，直到記錄顯示 `太棒了！日曆已經完全清空...` 為止。
